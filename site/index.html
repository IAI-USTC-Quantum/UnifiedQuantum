核心能力

一套框架，覆盖全链路

从线路构建到错误缓解，UnifiedQuantum 提供量子计算所需的完整工具链。

统一电路构建

一致的 API 构建量子线路，支持原生构建器及 Qiskit / Cirq 导入，输出 OriginIR 或 OpenQASM 2.0。

多平台后端

聚合 OriginQ、Quafu、IBM Quantum、QuarkStudio 四大量子云平台，一套代码切换后端。

本地仿真

状态向量与密度矩阵模拟，复用真实芯片拓扑数据的含噪模拟，无需云端凭证即可开发调试。

芯片校准

XEB 交叉熵基准测试，逐层门保真度测量，读出误差矩阵标定，结果自动缓存与 TTL 管理。

错误缓解 (QEM)

M3 矩阵无关测量误差缓解，线性求逆修正，结合校准数据自动应用于测量结果。

CLI & Gateway

功能完整的命令行工具 uniqc，以及 Web Gateway 可视化管理任务、查看结果。

快速上手

三分钟，从安装到运行

Python API 
from uniqc import Circuit

c = Circuit()
c.h(0).cnot(0, 1)
c.measure(0).measure(1)

result = c.simulate(shots=1024)
print(result.counts)
CLI 
uniqc simulate circuit.qasm --shots 1024

# 提交到真实量子芯片
uniqc submit circuit.ir \
  --platform originq \
  --backend WK_C180

# 查看任务结果
uniqc result <TASK_ID> --wait
